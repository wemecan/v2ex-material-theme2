# V2EX MATERIAL THEME

## 兼容性
	
	最新版 Chrome
	IE11
	
	如果要支持 Firefox，用这个：https://github.com/zdhxiong/v2ex-material-theme

## 代码

	@import url("//dn-startplay.qbox.me/v2ex-material-theme2/v2ex.min.css");
	
	
## 使用方法

进入 www.v2ex.com ，登录后进设置，在自定义CSS处填入代码保存即可。
